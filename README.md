## Setup and Commands

Use the following commands to work with this project:

```bash
# Install dependencies
$ npm install

# Run the development server
$ npm run dev

# Run the production build
$ npm run start

# Build for production
$ num run build

# Run unit tests with coverage
# $ npm run test --coverage
```
